Space Shooter


I built this game using HTML Canvas and Vanilla JavaScript for the frontend. In the backend I run my own Node JS application linked to my SQL database, both running on Heroku. Via Node I post and receive highscores from my database. I implemented gamepad support via gamepad API.

Hit multiple enemies in a row without missing a shot to get a score multiplier. 10 hits in a row doubles the points you earn. 20 hits give you a triple bonus. The game consists of three looping levels with increasing speed.
